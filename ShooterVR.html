Game Description

A VR-based shooting game for Android was developed using immersive 3D interactions and optimized rendering techniques.Integrated XR interaction inputs with responsive feedback to enhance realism and user engagement. Applied performance optimization strategies such as batching, texture compression, and efficient memory usage for smooth mobile VR experience.

Key Features & Technical Details

C# Programming

Clean, modular C# code with optimized game logic, object pooling, and efficient collision detection systems.

Dynamic Obstacles

Multiple enemy types with unique behaviors: enemies move towards the player, some at different speeds, some attacking directly, and others appearing in waves to increase difficulty.

Progressive Difficulty

Increasing speed and obstacle density over time, with power-up systems and score multipliers.

Visual Polish

Custom desert-themed artwork, particle effects for sand trails, smooth animations, and dynamic lighting.
